# TP : 1 - Todo list

## Installation
-  Cloner le repo
```
git clone https://github.com/zHatsuharu/VueJS-2024.git
```
-  Aller dans `TP1`
```
cd TP1/
```
-  Installer les modules node
```
npm i
```
-  Lancer le projet
```
npm run dev
```
-  Accéder au site `http://localhost:5173/`
